Sistem Informasi Inventory dengan Framework CodeIgniter

CI Inventory ialah sistem informasi stok data yang fully Responsive, Clean Design, Modern, Mudah untuk digunakan dan dikembangakan. Di buat dengan menggunakan HTML5 & CSS3, Bootstrap, Font Awesome, dan Framework Codeigniter.

RESPONSIVE LAYOUT

Sistem Informasi Inventory ini dibanguan dengan Template AdminLTE yang Responsive dan Modern.

BOOTSTRAP

Dibangun dengan menggunakan Bootstrap, sehingga tampil apik dan mudah untuk di kembangakan lagi.

Multi User

Sistem Informasi ini bisa diakses oleh beberapa user sekaligus, seperti Administrator, Manager dan Operator.

Software Features

Aplikasi ini bisa dijadikan referensi untuk Tugas Akhir dan Skripsi dan juga bagi anda yang mempunyai usaha di bidang pengolahan stok data.

Aplikasi ini bisa diakses melalui Smartphone anda, Anda bisa mengakses laporan dimanapun anda berada.

Anda akan mendapatkan source code dan database lengkap serta BONUS menarik.
